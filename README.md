# TrebuhS Tap

## How do I install these formulae?

`brew install trebuhs/tap/<formula>`

Or `brew tap trebuhs/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "trebuhs/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
